# Graphica
This project is my implementation of frame graph.

## Framegraph features
- [x] ID/Handle system for VK objects
- [x] Automatic barrier synchronization
- [x] integration with graphviz (kinda) 
- [ ] Transient resources (aliasing)
- [ ] Queue parallelization and synchronization
